Freepal Coin (FPC) - Whitepaper

1. Pendahuluan

Freepal Coin (FPC) adalah token kripto yang bertujuan untuk mendukung kebebasan finansial dan kesadaran sosial, terutama terhadap perjuangan rakyat Palestina...

2. Tujuan

  • Memberikan alat investasi terdesentralisasi
  • Membangun ekosistem digital pro-keadilan
  • Menjadi alat bantu mikro-transaksi di Freepal Miner

3. Tokenomics

Nama: Koin Freepal
Simbol: FPC
Jumlah Maksimum: 14.500.000 FPC
Blockchain: Polygon (MATIC)

4. Alokasi Token

  • 50% untuk komunitas (Freepal Miner / airdrop / reward)
  • 25% untuk pengembangan proyek
